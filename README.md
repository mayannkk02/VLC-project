## Project Description
This project is an audit and analysis of **VLC Media Player**, a popular free and open-source multimedia player developed by the VideoLAN project. The project includes both theoretical study and practical Linux shell scripting.

The report covers:
- Origin and philosophy of VLC
- GNU GPL licensing and open-source freedom
- Linux footprint of VLC
- Role of VLC in the FOSS ecosystem
- Comparison of VLC with proprietary media players

The practical part of the project includes **five shell scripts** related to Linux system inspection, VLC package analysis, storage auditing, log analysis, and open-source awareness.
